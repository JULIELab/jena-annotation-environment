Jena Annotation Environment
The Jena Annotation Environment(JANE) is a platform that supports the complete annotation life-cycle and allows for `focused' annotation based on active learning -- an intelligent selective sampling strategy that helps reduce the amount of data to be annotated substantially at almost no loss in annotation effectiveness.

Annotation GUI
The annotation GUI is part of JANE providing an intuitive GUI for annotators.
